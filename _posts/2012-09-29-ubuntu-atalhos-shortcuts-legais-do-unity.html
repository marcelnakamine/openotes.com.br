Minimizar todas as janelas: CTRL + SHIFT + D
Fechar Janela: ALT + F4
Executar comando: ALT + F2
Alternar entre janelas: ALT + TAB
Alternar entre janelas (ordem inversa): ALT + SHIFT + TAB
Mostrar todas as janelas abertas (Modo Expo): Super + W
Alternar entre os desktops: SHIFT + ALT + <SETA>
Mover a janela atual para outro desktop: CTRL + SHIFT + ALT + <SETA>

Bloquear tela: CTRL + ALT + L
Capturar a imagem da janela ativa: ALT + Print Screen
Capturar a imagem de uma área (o cursor vai virar uma cruz, clique e arraste para selecionar a área): SHIFT + Print Screen
