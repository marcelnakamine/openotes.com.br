 Para comemorar o 10˚ ano da Firefox, eles disponibilizaram uma atualização da linha Aurora, porém bem mais ousada, introduzindo o Firefox Developer Edition, "Construído para aqueles que constroem a Web".

 Uma primeira mudança para essa nova versão foi o Design escuro que segundo o Diretor Dave Camp (Director of Developer Tools at Mozilla) completa, que todas as ferramentas estão disponíveis no próprio navegador para que o desenvolvedor não precise alternar entre várias aplicações e fique focado para completar suas tarefas.
 Para agregar as inovações do browser, ele possui o WebIDE, que ajuda na construção de aplicações modernas da web e de aplicações de dispositivos móveis a partir de ferramentas bem úteis. E o Valence, antigo Firefox Tool Adapter. Com ele você pode desenvolver e depurar suas aplicações em diferentes dispositivos e navegadores.


 Para mais curiosidades, seus concorrentes também possuem uma versão beta. Segue alguns links abaixo:

  • Microsoft - Modern IE
